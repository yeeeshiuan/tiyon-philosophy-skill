# 十二長生對應體用狀態

**這是體用框架的狀態診斷工具，不是獨立系統。**

十二長生描述一個能量從生到滅再到重生的完整循環，對應體用在不同階段的狀態。

## 完整對應

### 體積累的方向（印在運作，根基在深化）

**絕** → **胎** → **養**

- **絕**：舊的用完全瓦解，但這也是轉化的前夜。體在最深處，還沒有形狀，但潛能在此。不是終點，是循環的轉折。
- **胎**：體在容裡孕育，還未成形。純粹的潛能狀態。印的滋養正在積累，外部幾乎看不見任何動靜。
- **養**：體被滋養，根基在積累。印右腦活躍，氣在充實。這個階段需要保護，不能過早用財左腦的邏輯介入。

### 體用開始展開（兩者協作，體仍為主）

**長生** → **沐浴** → **冠帶** → **臨官**

- **長生**：體開始自然展開，氣開始向外流動。用從體自然生出，方向感出現，但還很柔軟。
- **沐浴**：體初次接觸外部世界，脆弱但真實。這個階段容易受到外部標準的干擾，需要辨別哪些反饋是真實的，哪些是財左腦的雜訊。
- **冠帶**：體有了形，用開始成形。氣充實，欲有方向，可以開始借勢，但體還是主導的。
- **臨官**：體用開始協作，有方向但不失根。這是穩定發揮的狀態，用從體生出，勢為體所用。

### 臨界點（知止的關鍵時刻）

**帝旺**

- 用到極盛，表面上一切最好——影響力最大、成就最高、外部指標最亮眼。
- 但體已開始被消耗。用在最強的時刻，恰恰是體最需要被補充的時刻。
- **這是知止最關鍵的時刻**：帝旺繼續往前，就是衰的開始。
- 典型現象：股市大好但生育率探底（用的帝旺，體的衰）；事業頂峰但關係空洞；靈修商品化（形式帝旺，內容已死）。

### 用傷體的方向（財左腦過盛，需要回頭）

**衰** → **病** → **死** → **墓**

- **衰**：用過了帝旺，體的訊號開始浮現——疲倦感、空洞感、「這樣下去不對」的直覺。若此刻知止，還可以轉向。
- **病**：體明顯受損，用還在運作但已空轉。語言還在，形式還在，但那個從體生出的活力已不在。
- **死**：體與用的連結斷裂。用成為空殼——制度還在執行，但它服務的體已不在。外儒內法的極端形式。
- **墓**：沉澱，收藏。舊的用被封存，體在最深處等待。看起來什麼都沒有，但這是胎的前身。

## 診斷應用

**個人層次**：
- 感到空洞或意義危機 → 可能在衰病之間，用已脫離體
- 感到充實但說不出為什麼 → 可能在長生臨官，體用健康展開
- 外部成就最高但內在最不安 → 帝旺，需要知止

**組織層次**：
- 制度完善但執行者無力 → 冠帶過渡到病，形式有體已無
- 快速成長但人才流失 → 帝旺往衰，用消耗體

**社會層次**：
- 用的指標帝旺（GDP、股市）但體的訊號在衰（生育率、信任）→ 典型的帝旺陷阱

## 循環的意義

這個循環不是線性的失敗，而是體用自然的節律。

墓不是終點，胎才是——每一次死墓，都是下一次胎養的前身。

知止的能力，讓你在帝旺時感知到「該回頭了」，不需要走到死墓才被迫重來。

損之又損，以至於無為——主動選擇回到胎養，是比被動等到死墓更有智慧的路。
