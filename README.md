# 體用哲學思考框架 — Claude Code Skill

一個以**體用**為核心的中華哲學思考框架，作為 [Claude Code](https://claude.ai/code) 的自定義 Skill 運行。當你與 AI 進行任何決策思考、哲學性分析、社會現象解讀或個人處境反思時，此框架會自動引入中華哲學的思考視角。

---

## 框架概覽

### 核心：體用

> **體**是根基、內在狀態、讓一切得以發生的根源。  
> **用**是從體自然展開的行動、形式、制度、語言。

體用不是平等的兩極——體在先，用在後。用從體生出才是健康的；用一旦脫離體而獨自運作，就是病態的開始。

**三個快速診斷問題：**
1. **體在哪裡？** 這個人、組織、社會的根基是什麼？充實還是空洞？
2. **用在做什麼？** 當下的行動、制度、語言，是從體自然生出的，還是已在空轉？
3. **知止了嗎？** 有沒有感知到「到這裡了」的能力？

## 框架結構

### 主框架：體用（`references/TIYONG.md`）

體用的健康與病態對照：

| 狀態 | 描述 |
|---|---|
| **健康** | 體充實，用自然展開，用完回到體補充 |
| **病態一（中華歷史病）** | 用壓制體——以用代體，滿口道德仁義但體已空洞 |
| **病態二（西方歷史病）** | 體的缺席——以用充體，工具理性極度發達但不知為何而活 |

體用的自我背叛規律：任何活的洞見，一旦制度化，就會部分殺死洞見的活性。佛陀、耶穌、孔子、老子——莫不如此。

---

### 延伸框架一：氣欲勢容（`references/QIYUSHI.md`）

描述在**容**這個場域裡運作的能量層次：

| 概念 | 性質 | 西方對應 |
|---|---|---|
| **氣** | 體充實後自然溢出，乾之德 | Virtue（亞里斯多德）、Conatus（斯賓諾莎）|
| **欲** | 有方向的內在驅動，需要對象錨定 | Eros（柏拉圖）、Will to power（尼采）|
| **勢** | 純粹外在，借來用、用完還 | Fortune（馬基維利）、Capital（布迪厄）|
| **容** | 場域本身，坤之德，讓一切得以發生 | Lichtung（海德格）、量子場論的場 |

健康的流動：**氣充實 → 自然生欲 → 可以借勢 → 勢用完回到氣補充**

---

### 延伸框架二：十二長生（`references/SHIERZHANGSHENG.md`）

體用狀態的動態診斷工具，描述能量從生到滅再到重生的完整循環：

```
絕 → 胎 → 養 → 長生 → 沐浴 → 冠帶 → 臨官 → 帝旺 → 衰 → 病 → 死 → 墓 → （回到絕）
```

**帝旺**是知止最關鍵的時刻——用到極盛，外部指標最亮眼，但體已開始被消耗。典型現象：事業頂峰但關係空洞；股市大好但生育率探底。

---

### 延伸框架三：印右腦與財左腦、知止、指點（`references/YINZUO.md`）

**印右財左**：
- **財左腦**——邏輯、分析、量化，驗證方式是外部的（實驗、邏輯、市場回應）
- **印右腦**——整體感知、當下流動、無法命名，驗證方式是內部的（直接感知、身體回應）
- 問題不是二元對立，而是現代社會普遍用財左腦的方式去追求印右腦的滋養（KPI管理創意、正念APP量化冥想）

**知止**：感知到用的邊界在哪裡，不讓用脫離體而獨自運作。知止從印右腦生出，無法用財左腦計算。

**指點**：不是從外部給進去，而是感應到對方內部已有的東西只差一步，做最小的動作讓它浮現。西方最接近的對應：蘇格拉底的助產術。

## 觸發情境

此 skill 適用於討論以下問題時：

- 「夠不夠」、「該不該」——涉及知止與邊界感知
- 「為什麼感覺空」——體用失衡的個人診斷
- 「制度與人」——用壓制體的組織或社會分析
- 「個人與集體」——容的場域如何形成與崩潰
- 「動機與行動」——氣欲勢的能量層次診斷
- 「根基與展現」——體用健康與否的核心問題
- 任何系統性失衡——個人、組織、社會、文化層次皆適用

---

## 使用原則

1. **體用優先**：所有分析從體用關係切入
2. **知止為核心**：不讓用空轉，感知回到體的時機
3. **不急著給答案**：承載、感應、讓場域說話，而非快速給解法
4. **指點而非說教**：最小的介入，讓內在已有的東西自然浮現
5. **動態觀察**：用十二長生判斷當下狀態，而非靜態標籤
